# 군주온라인 정보 사이트

이 프로젝트는 군주온라인 정보 사이트 입니다.

## 만든 목적

군주온라인 정보를 공유하고 마을 주민 관리를 위한 홈페이지 입니다.

## 개발 기간

2025.02 ~

## 시작 가이드 

초기 다운로드

    $git clone https://github.com/youjuyong/project_Gunzu.git
    $npm install
    $npm start

## 개발 환경
node.js 버전 : v16.16.0

jre : jdk 17

## 개발 스택
FrontEnd : React, TypeScript, JavaScript

BackEnd : SpringBoot, JPA, MyBatis

## 배포
FrontEnd : AWS S3

BackEnd : AWS EC2
